ICT Fundamentals

This comprehensive course is designed to boost workplace productivity by equipping participants with a deep understanding and practical knowledge of various office productivity tools. Covering essential software such as word processors, spreadsheets, presentation tools, and databases, the course also delves into time management, data organization, and effective communication techniques.

Graphic Design

Learn how to create visual content that helps to communicate messages with an audience. The art of visual communication is essential, especially for brands looking to connect with their target audience, and graphic design is the perfect solution.

UI/UX Design

In this course, we'll cover the theory behind UI and UX design. You'll also design your own wireframes and interactive prototypes. Learning UI and UX basics can help you collaborate better on team projects and create new career opportunities.

Web Development

Web development, refers to creating, building, and maintaining websites that run online on a browser. It may, however, also include web design, web programming, and database management. The basic tools involved in web development are programming languages called HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript.
